# zzprompts Theme - Translation Files

This directory contains translation files (.po/.mo) for the zzprompts theme.

## How to Translate

1. Use a translation plugin like Loco Translate or Poedit
2. Text domain: `zzprompts`
3. All strings are wrapped in translation functions: `esc_html__()`, `_e()`, `_x()`

## Supported

The theme is ready for WPML and Polylang compatibility.

## Generating Translation Files

Use WordPress tools or plugins to generate .po and .mo files for your language.

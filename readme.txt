== Description ==

ZZ Prompts is a premium WordPress theme designed specifically for AI prompt libraries and collections. Built with modern glassmorphism design, performance optimization, and user experience in mind.

**Key Features:**

* Modern Glassmorphism Design with Dark Mode
* Custom Post Type for AI Prompts (via ZZ Prompts Core plugin)
* Taxonomies for Categories and AI Tools (ChatGPT, Midjourney, Claude, etc.)
* One-Click Copy to Clipboard functionality
* Like System with anti-spam protection
* Translation-ready with full RTL support
* SEO optimizations with JSON-LD Schema
* Fully responsive design
* Blog section with premium styling
* Monetization ready (Ad placements via Customizer)
* 5+ Page Templates (About, Contact, Login, etc.)
* 8 Custom Widgets for sidebar and footer
* Block Patterns for Gutenberg

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through 'Appearance > Themes' menu
3. Install the required "ZZ Prompts Core" plugin when prompted
4. Import demo content via Tools > Import
5. Configure theme settings via Customizer
6. Start creating prompts via 'Prompts' menu

== Frequently Asked Questions ==

= Does this theme require a plugin? =
Yes, the "ZZ Prompts Core" plugin is required for the Prompts custom post type and features. You'll be prompted to install it automatically.

= Is the theme translation-ready? =
Yes, all strings are translatable using the 'zzprompts' text domain. RTL languages are fully supported.

= Can I customize the colors? =
Yes, primary and accent colors can be changed via WordPress Customizer > ZZ Prompts Settings > Colors & Branding.

= Does it support dark mode? =
Yes, the theme includes automatic dark mode based on user's system preference, plus a manual toggle.

= How do I add ads? =
Go to Customizer > Ad Management to enable and configure ad slots for header, prompts, blog, and sidebars.

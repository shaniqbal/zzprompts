A high-performance AI Prompt Library theme - Translation-ready, secure, and optimized for copy/paste AI prompts with integrated blog section.

== Description ==

zzprompts is a standalone, classic WordPress theme designed specifically for AI prompt libraries and collections. Built with performance, security, and user experience in mind.

**Key Features:**

* Custom Post Type for AI Prompts
* Taxonomies for Categories and AI Tools (ChatGPT, Midjourney, etc.)
* One-Click Copy to Clipboard functionality
* Like System with LocalStorage (no login required)
* Translation-ready with full multi-language support
* SEO optimizations with JSON-LD Schema
* Mobile responsive
* Blog section with Sidebar layouts
* Monetization ready (Ad placements)

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through 'Appearance > Themes' menu in WordPress
3. Configure theme settings via Customizer
4. Start creating prompts via 'Prompts' menu

== Frequently Asked Questions ==

= Does this theme require a parent theme? =
No, zzprompts is a standalone classic theme.

= Is the theme translation-ready? =
Yes, all strings are translatable using the 'zzprompts' text domain.

= Can I customize the colors? =
Yes, via CSS variables in assets/css/main.css or through WordPress Customizer.

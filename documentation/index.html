Installation

Method 1: WordPress Admin (Recommended)

  1. Go to Appearance → Themes → Add New
  2. Click Upload Theme
  3. Choose the zzprompts.zip file
  4. Click Install Now
  5. After installation, click Activate

Method 2: FTP Upload

  1. Unzip zzprompts.zip on your computer
  2. Upload the entire zzprompts folder to /wp-content/themes/
  3. Go to Appearance → Themes and activate

Required Plugin

After activating the theme, you will see a notice: "This theme requires the following plugin: ZZ Prompts Core"

You MUST click "Begin installing plugins" to install this. It handles all the core features like Prompt Custom Post Type, Likes, and Views.

Demo Import

We've made it easy to get started with One Click Demo Import.

How to Import

  1. Ensure the One Click Demo Import plugin is installed (via TGM recommended plugins).
  2. Go to Appearance → Import Demo Data.
  3. Click the Import button.
  4. Wait for the process to finish (it may take 2-5 minutes).

Alternatively, you can manually import the XML file located in demo-content/demo-content.xml using the WordPress Importer.

Theme Setup

1. Set Up Menus

Go to Appearance → Menus. Create a menu and assign it to the Primary Menu location.

2. Configure Homepage

Go to Settings → Reading. Select "A static page" and choose your desired Homepage (imported from demo) as the "Homepage".

3. Setup Permalinks

Go to Settings → Permalinks and select Post name. Click Save Changes.

Customizer Options

All theme settings are located in Appearance → Customize → ZZ Prompts Settings.

Section Description
Homepage Toggle sections (Hero, Prompts, Blog, CTA), edit titles and subtitles.
Layout & Spacing Adjust header gaps, hero padding, and overall layout density.
Prompts Customize "Copy" button text, success messages, and meta visibility.
Colors Set your Primary and Accent brand colors. Dark mode is automatic.

Creating Prompts

Add a New Prompt

  1. Go to Prompts → Add New
  2. Enter the Title (e.g., "SEO Blog Post Generator")
  3. Add a description in the main editor.
  4. Crucial Step: Scroll down to the "Prompt Data" box and paste the actual AI prompt code there.
  5. Select a Category (e.g., Marketing) and AI Tool (e.g., ChatGPT).
  6. Click Publish.

Widgets

The theme comes with 8 custom widgets found in Appearance → Widgets.

  • ZZ: Brand & Social (Footer) - Logo and social links.
  • ZZ: Popular Prompts (Sidebar) - Top prompts by likes/views.
  • ZZ: Newsletter (Footer) - Email signup form.
  • ZZ: Category Tags (Sidebar/Footer) - Tag cloud for tools/categories.
  • ZZ: Author Bio (Sidebar) - Profile card with avatar.
  • ZZ: Ad Banner (Sidebar) - Custom ad code placement.

Page Templates

When creating a new page, verify the Template setting in the Page Attributes box.

  • Default Template: Standard page with container.
  • About Us: Custom layout for company info.
  • Contact Us: Includes FAQ and contact form area.
  • Login / Forgot Password: Custom styled auth pages.

Ad Management

Monetize your site easily via Customize → Ad Management.

You can paste Google AdSense or any custom HTML/JS code into these slots:

  • Header (Global)
  • Below Prompt Code (Single Prompt)
  • Archive Grid (In-feed native ads)
  • Blog Post (Top/Bottom)

Translation (I18n) & RTL

The theme is fully translation-ready and supports RTL languages (Arabic, Hebrew, etc.) out of the box.

How to Translate

  1. Locate the prompts.pot file in the /languages/ folder.
  2. Use a tool like Poedit or the Loco Translate plugin.
  3. Create a new translation for your language (e.g., fr_FR).
  4. Translate the strings and save.
